# Tic Tac Toe Fullstack (Nakama + React)

Realtime multiplayer Tic Tac Toe built with:

- **Backend:** [Nakama](https://heroiclabs.com/docs/nakama/) + PostgreSQL + JavaScript runtime module
- **Frontend:** React + `@heroiclabs/nakama-js`
